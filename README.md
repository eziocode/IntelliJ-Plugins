# IntelliJ Plugins by Aswin

A collection of productivity-enhancing plugins for JetBrains IDEs.

## Plugins

### Git Assume Unchanged

Easily manage Git's `assume-unchanged` flag directly from IntelliJ IDEA and all JetBrains IDEs.

**Features:**
- Mark files as assume-unchanged to ignore local changes
- Remove assume-unchanged flag to resume tracking
- Multi-file support
- Works in all Git-related contexts
